indexOf()

Phương indexOf()thức trả về chỉ mục (vị trí) lần xuất hiện đầu tiên của một chuỗi trong một chuỗi

lastIndexOf()

Phương lastIndexOf()thức trả về chỉ mục của lần xuất hiện cuối cùng của một văn bản đã chỉ định trong một chuỗi

Cả hai indexOf()và lastIndexOf()trả về -1 nếu không tìm thấy văn bản Cả hai phương pháp đều chấp nhận tham số thứ hai làm vị trí bắt đầu tìm kiếm Các lastIndexOf()phương thức tìm kiếm ngược lại (từ cuối đến đầu), nghĩa là: nếu tham số thứ hai là 15, tìm kiếm bắt đầu ở vị trí 15 và tìm kiếm đến đầu chuỗi

String search()

The search() method searches a string for a string (or a regular expression) and returns the position of the match

The search() method cannot take a second start position argument. The indexOf() method cannot take powerful search values (regular expressions)

String match()

The match() method returns an array containing the results of matching a string against a string (or a regular expression)

String matchAll()

Phương matchAll()thức trả về một iterator chứa kết quả so khớp một chuỗi với một chuỗi (hoặc một biểu thức chính quy)

String includes()

Phương includes()thức trả về true nếu một chuỗi chứa một giá trị được chỉ định. Nếu không thì nó trả về false

String startsWith()

Phương startsWith()thức trả về true nếu một chuỗi bắt đầu bằng một giá trị được chỉ định. Nếu không thì nó trả về false

String endsWith()

Phương endsWith()thức trả về true nếu một chuỗi kết thúc với một giá trị được chỉ định. Nếu không thì nó trả về false
